# Simple To‑Do List (Простой список дел)

Мини‑проект подходит для изучения основ веб‑разработки: работа с DOM, событиями и `localStorage` в JavaScript.

## Возможности

Приложение позволяет:

- добавлять задачи по нажатию Enter или кнопки «Добавить»;
- отмечать задачи как выполненные (визуально отображается зачёркиванием);
- удалять отдельные задачи;
- сохранять список задач между перезагрузками страницы (данные хранятся в `localStorage`).

## Инструкция по запуску

1. Клонируйте репозиторий:

   ```bash
   git clone <URL-репозитория>
   ```

2. Перейдите в папку проекта:
   ```bash
   cd todo-js
   ```
3. Откройте файл index.html в любом современном браузере.
4. Начните добавлять задачи!

## Структура проекта

index.html — разметка страницы (HTML5);

style.css — стили оформления (CSS3);

script.js — логика приложения (JavaScript).

## Как работает сохранение

Данные сохраняются в localStorage браузера:

при добавлении/удалении задачи список обновляется в хранилище;

при загрузке страницы приложение считывает данные из localStorage;

если данных нет, создаётся пустой список.

## Для кого подходит

Проект идеально подходит для:

новичков в веб‑разработке, изучающих чистый JavaScript;

тех, кто хочет попрактиковаться в работе с DOM;

желающих разобраться в механизме localStorage;

студентов, выполняющих учебные задания по фронтенду.
